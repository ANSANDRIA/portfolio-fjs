# Portfolio FJS

## UAS Pemrograman Web

Website Portfolio Personal menggunakan PHP Native dan MySQL.

## Fitur

- Login Admin
- CRUD Profile
- CRUD Skills
- CRUD Portfolio
- Contact Form
- Dashboard Admin
- Upload Foto
- Upload Gambar Portfolio

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- HTML
- CSS
- JavaScript

## Cara Menjalankan

1. Install XAMPP
2. Copy folder project ke htdocs
3. Import database `database/db_portfolio_fjs.sql`
4. Jalankan Apache dan MySQL
5. Buka browser

```
http://localhost/portfolio-fjs
```
